# Turn-based-game
Turn based game source files, made in Godot as a personal project.

# How to use
1. Download and install Godot (version 4.5 recommended)
2. Clone this repository
3. Open the project in Godot
4. Run it

## tech
Engine: Godot 4.5
Language: GDScript
Desired platform: PC
